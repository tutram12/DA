# Diplomarbeit  
## Enzi, Martinak, Tuttner  
  
### Idee
Raumklimamessgerät  
  
### Was soll das Objekt können?
Es soll die Lautstärke, den CO2-Gehalt und die Temperatur messen.  
Diese Werte sollen vor Ort verarbeitet und über ein Display am Gerät ausgegeben werden.
